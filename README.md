# Homelab
This is a documentation of my experiences with building a home server with old devices or self built machines.

I've always been a big enthusiast in exploring devices that are considered outdated to better understand the hardware that came with it. The moment I was able to gather enough money to build my own personal computer, I researched everything that I can about the installation and the possiblities that I can use the device for, even if my initial goal was to just play some games with my friends.
